DroneVis - Visualização de dados de drones brasileiros

Visualizando o cenário brasileiro de drones

Objetivos:

  • Estudo da utilizaçãod e drones no Brasil
  • Estudo da distribuição de drones com relação aos estados, proprietários e ramos de utilização
  • Analisar os padrões e tendências que sirvam como base para o planejamento de uma infraestrutura de Internet dos Drones Brasileira

Alunos:

  • Alisson Renan Svaigen
  • Lailla Milainny Siqueira Bine

Base de dados:

  • Drones oficialmente cadastrados pela ANAC
  • Base de dados pública
  • Dados disponíveis:
    • Data de inscrição e validade;
    • Dados do operador (físico e jurídico);
    • Dados do fabricante e do modelo;
    • Dados de localização (cidade e estado);
    • Ramo de atuação.
